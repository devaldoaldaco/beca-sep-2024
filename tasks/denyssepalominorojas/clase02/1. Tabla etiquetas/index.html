Etiquetas de HTML

Etiqueta Descripcion
!DOCTYPE Indica la version de HTML que se está utilizando en la página
meta Etiqueta que define metadatos para resumir el contenido de la página y facilitar que sea entendida por los buscadores y navegadores web
title Etiqueta que definen el titulo de nuestra página web que aparece en la pestaña del navegador
link Etiqueta que enlaza hojas de estilo externas
html /html Etiqueta que define el inicio y el final de la página web
body Etiqueta que contiene todo el contenido visible de la página
h1 - h6 Etiquetas que definen los diferentes tamaños de encabezados, títulos, subtítulos, etc.
a Etiqueta que define los enlaces en los que el usuario puede hacer clic para enviarlo a otra página web u otra sección diferente
img Etiqueta que define una imagen que se mostrará en la página web
map Etiqueta que define un mapa de imágenes
area Etiqueta que define una región en un mapa de imágenes
p Etiqueta que define un parrafo
b Etiqueta que da formato de negrita a cualquier texto que se encuentre dentro de esta
i Etiqueta que da formato de cursiva a cualquier texto que se encuentre dentro de esta
u Etiqueta que da formato de subrayado a cualquier texto que se encuentre dentro de esta
address Etiqueta para indicar información de contacto
style Etiqueta que contiene estilos CSS internos
script Etiqueta que contiene scripts de JavaScript
br Etiqueta que inserta un salto de línea
hr Etiqueta que inserta una línea horizontal
strong Etiqueta que enfatiza el texto (negrita)
em Etiqueta que enfatiza el texto (cursiva)
small Etiqueta que más pequeño el tamaño del texto
mark Etiqueta que resalta el texto
del Etiqueta que tacha el texto
ins Etiqueta que subraya el texto
ul Etiqueta que define una lista no ordenada
ol Etiqueta que define una lista ordenada
li Etiqueta que define los elementos de una lista
dl Etiqueta que define una lista descriptiva
dt Etiqueta que define un término de la lista descriptiva
dd Etiqueta que da detalles sobre el término de la lista descriptiva
table Etiqueta que define una tabla
caption Etiqueta que define el título de una tabla
thead Etiqueta que define el o los encabezados de una tabla
tbody Etiqueta que define el cuerpo de una tabla
tfoot Etiqueta que define el pie de una tabla
tr Etiqueta que define una fila de una tabla
th Etiqueta que define una celda como encabezado
td Etiqueta que define una celda de una tabla que contiene datos
form Etiqueta que define un formulario
input Etiqueta que crea controles interactivos para formularios
textarea Etiqueta que crea un cuadro de texto de varias líneass en una página web
button Etiqueta que define un botón
select Etiqueta que define menús desplegables
option Etiqueta que representa un elemento dentro de un select
label Etiqueta que representa una etiqueta explicativa
fieldset Etiqueta que agrupa elementos de un formulario
legend Etiqueta que define una leyenda para un fieldset
legend Etiqueta que define una leyenda para un fieldset
