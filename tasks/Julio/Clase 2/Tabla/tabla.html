Etiquetas de html:

Etiqueta Descripción
head definir valores en la cabecera del documento, pueden ser meta datos, el título o links a archivs externos
body define el cuerpo en el documento html
table crear una tabla
!DOCTYPE html indica que el documento trabaja con la última versión de html
title Define el título de la página
meta definición de metadatos
style definir estilos en el mismo documento html
a indicar un enlace que puede hacer redirección hacia otra página
h1,h2,h3,h4,h5,h6 Definen encabezados de distintos tamaños
img etiqueta para agregar una imagen al documento
p Definición de un parrafo
b el texto contenido entre la apertura y el cierre de esta etiqueta se mostrará en negritas
link enlazar recursos externos al html
nav indicar el contenido que contendrá la sección de navegación de la página
li definición de una lista
header Se usa para definir la cabecera de lo que se mostrará en la página
footer Define el contenido del pie de página
main Definir el contenido principal de la página, solo debe haber uno
section Definir una sección agrupada de contenido
form Definición de un formulario
input Definición de un campo para entrada de información de parte del usuario
article pequeño componente de información apartada del resto
aside contenido indirectamente relacionado al contenido principal
canvas etiqueta para realizar gráficos o incluso animaciones
abbr Contiene un acrónimo y se hace uso de un atributo "title" para dejar una descripción
address se puede usar para agrupar la información de contacto
area Define un área dentro de un map con partes que se pueden clickear
base especifica una url base para los elementos en el documento. Los enlaces en la página buscarán su recurso en esa url base
bdi aisla un texto que puede estar en un formato diferente
bdo especifica la dirección del texto
blockquote agrupa un texto que es citado de alguna fuente
br Inserta un salto de línea
button agrega un botón clickeable en el documento
caption inserta una pequeña descripción para una tabla
cite sirve para diferenciar un texto de otro, para denotar que es importante
code le da estilos al texto encerrado para que parezca código de un programa
col Especificar propiedades para una columna o un grupo de ellas
colgroup Agrupar etiquetas "col"
data Asocia el contenido de la etiqueta con un valor numérico
datalist Contiene la lista de opciones disponibles para algunos controles
dl engloba una lista de descripciones
dt especifica el título de una definición dentro de una dl
dd especifica la descripción de una definición dentro de una dl
details Esconde la información adicional para un tèrmino. Al dar click en el término, aparecen los detalles
div contenedor genérico para encerrar contenido
em agrega énfasis en el texto que abarca
embed punto de integración para una aplicación externa
fieldset Agrupa campos relacinados de un formulario
iframe Permite insertar una página html dentro de otra
ins Señala la parte de un texto que ha sido insertado, se usa con del
label Representa una etiqueta para un element en la interfaz de usuario, se asocia al mismo
map para crear una imagen o animación con partes que se pueden clickear
meta describe metadata en un documento html
noscript Tiene el contenido que debe mostrar el navegador cuando no soporta los scripts
ol agrupa los elementos de una lista ordenada
option representa un item dentro de un select
output campo para mostrar el resultado de un cálculo
picture contiene distintas imagenes que se mostrarán de acuerdo con las condiciones de la interfaz
progress literalmente pinta una barra sobre el progreso de una tarea
q marca citas cortas
s Tacha el texto con una línea horizontal
script sirve para agregar código de javascript, ya sea ahí mismo o especificar una fuente externa
select Control que representa un menú de opciones
span sirve para abarcar elementos en linea
strong poner fuerte énfasis en un texto
summary el texto de un elemento details que va a guardar su descripción
textarea Control para la edición multilinea de texto sin formato
time representa un periodo específico en el tiempo
u subraya el texto abarcado
var muestra las variables que son parte de una expersión, usualmente en italica
video Para insertar un video en el documento
wbr parte del texto donde puede ocurrir un salto de línea dependiendo como cambia la condición de la interfaz
