Etiquetas HTML
Etiqueta Definición Atributos principales
!DOCTYPE Tipo de documento (versión de html empleada)
html Representa el inicio y fin de nuestro archivo html lang
< !-- ... -- > Comentario (sólo visible en el código fuente)
base Especifica la direccion URL base que se utilizara href, target
head Provee meta datos acerca del documento
title Título del documento (se muestra en la pestaña del navegador)
link Enlace a otros archivos (hoja de estilo, etc.) href, rel, media, type, title
meta Usada para definir metadatos que no se pueden definir en la etiqueta HTML name, content, charset
style Etiqueta para poder introducir codigo CSS type, title
body delimita el cuerpo del documento
article articulo
section seccion
nav navegacion
aside lateral
h1 a h6 encabezado de nivel 1 a 6
hgroup grupo de encabezados
header cabecera
footer pie de la pagina
address direccion (informacion sobre el autor)
