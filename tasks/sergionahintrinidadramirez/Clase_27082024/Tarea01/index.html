Tipos de Etiquetas en HTML

Tipo de Tag Descripción Ejemplo Descripción de Ejemplos
Tags de Estructura Define la estructura de la página web. <html>, <head>, <body>, <title>
  1. <html>: Define el inicio y el final de la página web.
  2. <head>: Se le considera el encabezado de la página que incluye las metaetiquetas y archivos externos.
  3. <body>: Es la sección donde están todos los elementos que se mostrarán.
  4. <title>: Es el texto que aparecerá en la pestaña del navegador.
Tags de Metadatos Proporciona información sobre el documento. <meta>, <base>, <link>, <style>
  1. <meta>: Aporta información de la página a la web.
  2. <base>: Especifica la dirección URL base para todas las direcciones.
  3. <link>: Agrega elementos externos como archivos .css para dar estilo a la página.
  4. <style>: Se encarga de dar estilo a las etiquetas directamente en el HTML.
Tags de Encabezados Define los encabezados de las secciones del documento. <h1> a <h6>
  1. <h1-6>: Son los títulos o encabezados, mientras más pequeño el número, más grande el tamaño del texto.
Tags de Texto Define el contenido de texto dentro del documento. <p>, <br>, <span>, <strong>, <em>
  1. <p>: Es la etiqueta para generar un párrafo.
  2. <br>: Realiza un salto de línea.
  3. <span>: Sirve para aplicar estilos a partes específicas del texto sin alterar su estructura.
  4. <strong>: Resalta en negritas el texto.
  5. <em>: Resalta el texto en cursiva.
Tags de Listas Define listas de elementos, ya sean ordenadas o no ordenadas. <ul>, <ol>, <li>, <dl>, <dt>, <dd>
  1. <ul>: Crea una lista no ordenada, los elementos tienen un punto como marcador.
  2. <ol>: Crea una lista ordenada, los elementos tienen números o letras como marcadores.
  3. <li>: Define un elemento de lista dentro de <ul> o <ol>.
  4. <dl>: Crea una lista de definiciones.
  5. <dt>: Define un término en una lista de definiciones.
  6. <dd>: Define la descripción de un término en una lista de definiciones.
Tags de Enlaces Define enlaces a otros recursos o documentos. <a>, <nav>
  1. <a>: Crea un enlace a otra página o recurso.
  2. <nav>: Define una sección de navegación que contiene enlaces.
Tags de Imágenes y Multimedia Define elementos multimedia como imágenes, videos y audio. <img>, <video>, <audio>, <source>, <track>
  1. <img>: Inserta una imagen en la página.
  2. <video>: Inserta un video que el usuario puede reproducir.
  3. <audio>: Inserta un archivo de audio que el usuario puede reproducir.
  4. <source>: Especifica múltiples fuentes para elementos multimedia como <video> o <audio>.
  5. <track>: Proporciona subtítulos o descripciones de texto para medios como videos.
Tags de Formularios Define formularios y elementos interactivos para la entrada del usuario. <form>, <input>, <textarea>, <button>, <select>, <option>, <label>
  1. <form>: Crea un formulario para que el usuario ingrese datos.
  2. <input>: Define un campo donde el usuario puede ingresar información.
  3. <textarea>: Crea un campo de texto de varias líneas.
  4. <button>: Crea un botón que el usuario puede hacer clic.
  5. <select>: Crea un menú desplegable de opciones.
  6. <option>: Define una opción dentro de un menú desplegable.
  7. <label>: Asocia una etiqueta con un elemento de formulario, facilitando la selección del elemento.
Tags de Tablas Define la estructura y contenido de una tabla. <table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot>
  1. <table>: Crea una tabla.
  2. <tr>: Define una fila dentro de la tabla.
  3. <td>: Define una celda dentro de una fila.
  4. <th>: Define una celda de encabezado, generalmente en negritas.
  5. <thead>: Agrupa el contenido del encabezado de la tabla.
  6. <tbody>: Agrupa el cuerpo principal de la tabla.
  7. <tfoot>: Agrupa el pie de la tabla.
Tags de Estilo Define estilos o apariencia para los elementos del documento. <style>, <link>, <div>, <span>
  1. <style>: Se usa para escribir código css dentro del HTML.
  2. <link>: Conecta archivos css externos para dar estilo a la página.
  3. <div>: Agrupa contenido y le da estilo a través de css.
  4. <span>: Aplica estilos a partes específicas de texto sin afectar el diseño general.
Tags Semánticos Define la estructura semántica del contenido. <header>, <footer>, <article>, <section>, <aside>, <main>, <nav>
  1. <header>: Define la cabecera o encabezado de una página o sección, generalmente incluye logotipos, menús de navegación, y otros elementos de introducción.
  2. <footer>: Define el pie de página de un documento o sección, generalmente contiene información de autor, enlaces relacionados, o derechos de autor.
  3. <article>: Representa una pieza independiente de contenido, como un artículo de una revista o blog.
  4. <section>: Agrupa contenido relacionado dentro de una página, como una sección de un capítulo o un bloque temático.
  5. <aside>: Define contenido adicional o complementario, como una barra lateral con enlaces relacionados o anuncios.
  6. <main>: Especifica el contenido principal de un documento, diferenciándolo del contenido que es repetitivo o común, como menús de navegación.
  7. <nav>: Define un conjunto de enlaces de navegación, generalmente utilizado para menús y barras de navegación.
Tags de Scripts Define y ejecuta scripts dentro del documento. <script>, <noscript>
  1. <script>: Inserta y ejecuta código JavaScript dentro de la página.
  2. <noscript>: Muestra contenido alternativo si el navegador no soporta JavaScript o lo tiene deshabilitado.
