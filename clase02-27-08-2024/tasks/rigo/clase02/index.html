Etiquetas HTML

Etiqueta Descripción
<html> Representaciòn de que es un documento html.
<a> Enlace para navegar a otras páginas.
<abbr> Abreviatura o acrónimo con su definición.
<address> Información de contacto del autor o propietario.
<area> Región activa dentro de una imagen map.
<article> Contenido independiente y autónomo.
<aside> Contenido relacionado pero separado del principal.
<audio> Reproductor de audio incrustado.
<b> Texto en negrita sin importancia semántica.
<base> URL base para los enlaces relativos.
<bdi> Aisla el texto que podría cambiar de dirección.
<bdo> Controla la dirección del texto.
<blockquote> Cita de un bloque de texto.
<body> Cuerpo del documento HTML.
<br> Salto de línea en el texto.
<button> Botón interactivo para enviar o ejecutar acciones.
<canvas> Área para gráficos dibujados por JavaScript.
<caption> Título para una tabla.
<cite> Referencia a una fuente o obra.
<code> Fragmento de código en formato monoespaciado.
<col> Define propiedades de columnas en una tabla.
<colgroup> Grupo de columnas con propiedades comunes en una tabla.
<data> Asociar un valor de datos con un contenido.
<datalist> Lista de opciones predefinidas para un campo de entrada.
<dd> Definición de un término en una lista de definición.
<del> Contenido eliminado del documento.
<details> Información adicional que el usuario puede mostrar u ocultar.
<dfn> Define un término en su contexto de definición.
<dialog> Cuadro de diálogo interactivo.
<div> Contenedor genérico para agrupar contenido.
<dl> Lista de definición de términos y sus descripciones.
<dt> Término en una lista de definición.
<em> Énfasis en el texto, generalmente en cursiva.
<embed> Inserción de contenido externo, como multimedia.
<fieldset> Agrupa controles relacionados en un formulario.
<figcaption> Título para un elemento <figure>.
<figure> Contenido relacionado con una imagen o ilustración.
<footer> Pie de página para un documento o sección.
<form> Contenedor para los controles de entrada del usuario.
<h1> a <h6> Encabezados de diferentes niveles de jerarquía.
<head> Contenido del encabezado del documento HTML.
<header> Encabezado de un documento o sección.
<hgroup> Agrupa varios encabezados relacionados.
<hr> Línea horizontal para separar contenido.
<html> Elemento raíz de un documento HTML.
<i> Texto en cursiva sin importancia semántica.
<iframe> Marco en línea para incrustar otro documento.
<img> Imagen incrustada en la página.
<input> Campo de entrada para formularios.
<ins> Contenido insertado en el documento.
<kbd> Entrada del teclado o secuencia de teclas.
<label> Etiqueta para un control de formulario.
<legend> Título para un <fieldset>.
<li> Elemento de una lista.
<link> Relación con un recurso externo, como una hoja de estilo.
<main> Contenido principal del documento.
<map> Define un mapa de imagen interactivo.
<mark> Resalta texto importante.
<meta> Información metadata sobre el documento.
<meter> Medidor de un valor dentro de un rango conocido.
<nav> Enlaces de navegación dentro del documento.
<noscript> Contenido alternativo para navegadores sin soporte de JavaScript.
<object> Inserción de contenido multimedia o documento externo.
<ol> Lista ordenada.
<optgroup> Grupo de opciones en un menú desplegable.
<option> Opción dentro de un menú desplegable.
<output> Resultado de un cálculo o acción.
<p> Párrafo de texto.
<param> Parámetro para un elemento <object>.
<picture> Contenedor para diferentes versiones de una imagen.
<pre> Texto preformateado con espacios y saltos de línea conservados.
<progress> Indicador de progreso de una tarea.
<q> Cita en línea.
<rp> Marca alternativa para navegadores que no soportan <ruby>.
<rt> Texto de anotación para <ruby>.
<ruby> Anotación para caracteres de Ruby en textos.
<s> Texto tachado, desactualizado o no válido.
<samp> Muestra salida de ejemplo de un programa.
<script> Código JavaScript incrustado o enlazado.
<section> Sección del documento con un encabezado.
<select> Menú desplegable para seleccionar opciones.
<small> Texto más pequeño que el texto circundante.
<source> Fuente de medios para <audio> o <video>.
<span> Contenedor genérico en línea para estilizar texto.
<strong> Texto con fuerte énfasis, generalmente en negrita.
<style> Reglas CSS internas para el documento.
<sub> Texto subíndice.
<summary> Resumen de un elemento <details>.
<sup> Texto superíndice.
<table> Contenedor para una tabla.
<tbody> Cuerpo de una tabla.
<td> Celda de datos en una tabla.
<template> Plantilla para contenido que no se renderiza directamente.
<textarea> Área de texto multilínea en un formulario.
<tfoot> Pie de una tabla.
<th> Encabezado de una celda en una tabla.
<thead> Encabezado de una tabla.
<time> Representación de una fecha u hora.
<title> Título del documento mostrado en la pestaña del navegador.
<tr> Fila de una tabla.
<u> Texto subrayado, generalmente para énfasis.
<ul> Lista desordenada.
<var> Variable en un fragmento de código.
<video> Reproductor de video incrustado.
<wbr> Sugerencia de lugar para un salto de línea opcional.
