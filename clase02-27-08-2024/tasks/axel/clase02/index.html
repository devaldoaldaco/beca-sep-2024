Etiqueta Descripcion
!DOCTYPE Le dice al navegador que version de html es
html Tiene todo el contenido del documento html
Metadatos
head Contiene metadatos
title Define el titulo del sitio web hasta arriba
meta Define los metadatos(datos sobre los datos)
link Vincula otros archivos
script Define un script en JS
style Cointiene los estilos de CSS
Tags para estructurar html
body Todo el contenido visible de la pagina web
header Es el encabezado de un documento
nav Es el contenido que va a estar e n la navegacion
section Es una seccion del documento
article muestra el documento independiente como un articulo
aside Se define contenido adicional de poca importancia
h1-h6 Se utilizan para jerarquizar
footer Establece el pie de pagina
Tags para la agrupacion de contenido
p Etiqueta para escribir parrafos de texto
hr Es utilizada como separador
pre Se utiliza para agregar texto, manteniendo el formato del propio texto
blockquote Se usa para indicar el texto citado
ol Crea una lista ordenada
ul Crea una lista desoredenada
li Agrega un contenido a la lista
dl Cre una lista de definiciones
dt Representa un termino definido por la siguiente etiqueta
dd Representa la descripcion de un termino en la lista
figure Señala una figura como partel del html
figcaption Define la lyenda de una figura
div Contenedor para agrupar
Tags para la semantica del texto
a Se utiliza para crear hiperenlaces
strong Define como importante el texto
small Se utiliza para dejar un comentario
cite Define el titulo de una obra
sub Representa un subindice
mark Resalta el texto
span No tiene un significado especifico
br Crea un salto de linea
Tags para contenido
img Inserta un imagen
iframe Anida otro documento html
embed Usada para agregar contenido interactivo externo a html
object Se utiliza para ser procesado por un plugin
video Se usa para reproducir video en la pagina web
audio Se usa para cargar audio
svg Se llama a una imagen vectorizada
Tags para crear tablas
table Genera una tabla
caption Sirve para indicar el titulo de una tabla
Colgroup Sirve para agrupar dos o mas columnas
tbody Se usa para describir los datos de una tabla
thead Indica el bloque de las filas que describen a las columnas
tfoot Señala los bloques de las filas
tr Indica una fila de celdas de una tabla
td Define una celda de una tabla
th Define el encabezado de una celda
Tags para la creacion de formularios
form Tag de apertura y cierre del formulario
fieldset Conjunto de los elementos de los formularios
legend Indica el titulo del fieldset
label Define el nombre de un control del formulario
Input Hace el campo de ingreso de los datos para un formulario
button Representa un boton en el formulario
select Permite seleccionar entre un conjunto de opciones
textarea Añade un campo para que el usuario pueda agregar texto
